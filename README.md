# HackToHorizon2 - Système de Gestion d'Inventaire

Système interactif de gestion d'inventaire avec visualisation sur plan d'étage.

## Démarrage Rapide

### Installation

```bash
npm install
```

### Utilisation

Ouvrir `public/index.html` dans votre navigateur.

## Structure du Projet

```
HackToHorizon2/
├── src/              Code source
├── public/           Interface web
├── assets/           Images et ressources
├── data/             Fichiers de données CSV/XLSX
└── docs/             Documentation complète
```

## Fonctionnalités

- Gestion complète de l'inventaire avec filtres
- Visualisation interactive sur plan d'étage
- Statistiques détaillées
- Recherche rapide
- Interface responsive
